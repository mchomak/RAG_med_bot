HELIOCART RETAIL INTELLIGENCE (HRI)
Версия документа: 1.0
Дата: 12 января 2026

1. Профиль компании
HelioCart Retail Intelligence - продуктовая компания, которая поставляет SaaS-платформу для управления выручкой и операционными KPI в рознице.
Фокус: данные из касс (POS), складов (WMS), программ лояльности и онлайн-каналов, собранные в единую витрину метрик для категорийных менеджеров и директора по операциям.

2. Линейка продуктов
2.1 HelioPulse Dashboard
- Единое окно для KPI: выручка, маржа, оборачиваемость, out-of-stock, выполнение планов.
- Частота обновления данных: каждые 15 минут (near-real-time).
- Сегментация: по магазину, городу, категории, поставщику, SKU.
- Авто-алерты: падение маржи > 2 п.п. за сутки или рост out-of-stock > 5% по категории.

2.2 ShelfSense API
- API для интеграции с BI/ERP и витринами данных (REST + Webhooks).
- Нормализует справочники SKU и поставщиков, устраняет дубли через правила match-merge.
- Встроенные "витрины": DailySales, StockSnapshot, PromoEffectiveness.

2.3 PromoLab
- Модуль оценки эффективности промо и скидок.
- Метрика инкремента (uplift) строится в разрезе "контрольная группа vs. тест" по магазину и времени.
- Авто-рекомендации: остановить промо, если uplift < 1.5% при падении маржи > 0.7 п.п.

3. Технологические особенности (коротко)
- Хранилище: PostgreSQL + партиционирование по дате, горячие агрегаты в Redis.
- Коннекторы: 1С (выгрузка по регламенту), CSV, Google Sheets, вебхуки кассы.
- Безопасность: SSO (SAML/OAuth2), аудит действий, IP allow-list.

4. Коммерческие условия (тестовые, для примера)
- Тариф "Start": до 5 магазинов, 3 дашборда, без API.
- Тариф "Growth": до 50 магазинов, API + webhooks, 20 дашбордов.
- Тариф "Enterprise": без лимитов магазинов, отдельный контур, SLA и выделенный CSM.
